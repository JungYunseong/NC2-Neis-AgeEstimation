# :iphone: Project Title

### AI 나이측정 테스트, How Old Am I

## Key Features

* 얼굴 인식
  - 선택된 이미지에서 얼굴만 따로 추출을 합니다.
* 나이별 이미지 분류
  - CreateML로 학습된 모델로 추출된 얼굴 이미지를 나이별로 분류합니다.
